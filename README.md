# Netflix Data Analysis (EDA Project)

##  Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to uncover insights about movies and TV shows available on Netflix.

The goal is to analyze content distribution, trends over the years, country-wise production, ratings, and genres.

---

##  Objective
- Analyze the distribution of Movies vs TV Shows
- Identify the most common genres
- Find which country produces the most content
- Analyze content trends over the years
- Understand rating categories

---

##  Dataset Information
- Dataset Name: Netflix Movies and TV Shows
- Source: Kaggle
- Features Included:
  - title
  - director
  - cast
  - country
  - release_year
  - rating
  - duration
  - listed_in (genre)
  - type (Movie/TV Show)

---

## Tools & Technologies Used
- Python 
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

##  Steps Performed
1. Data Loading
2. Data Cleaning (handling missing values)
3. Data Transformation
4. Univariate Analysis
5. Bivariate Analysis
6. Visualization

---

##  Key Insights
- Majority of content on Netflix are Movies.
- The United States produces the highest content.
- Drama and International Movies are the most common genres.
- Content release increased significantly after 2015.
